# NEXVIA

Tu tiempo y destino son importantes.

## Problema

Los usuarios no saben cuándo llegará el transporte.

## Solución

NEXVIA permite:

- Consultar rutas
- Ver ubicación del bus
- Consultar tarifas
- Recargar saldo
- Obtener recompensas
